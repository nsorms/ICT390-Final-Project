Operating Systems

System softwares that manages computer hardware and software resources and provides common services for computer programs.

Linux Operating System

Linux is an open source operating system built around a common kernel. There are many different types of Linux distributions. If you are looking to build a desktop environment, GNOME and KDE are the two best Linux distributions you can use.

Linux allows a user to create and edit the environment more than most any other operating systems. Linux's code may be modified to the user's preferance.

Linux is a good choice for programming intensive and computer savy people, if you are looking to become a beteter computer expert give Linux a try!

Microsoft Windows Operating System

Microsoft Windows, or Windows, is a desktop environment. Windows was one of the first GUIs or Graphical User Interfaces, to hit the market.

Windows allows the user to customize some items of the interface but unless one get's Windows Professional, which comes with more features for those who know what they are doing.

Windows is widely used and a very common operating system. If you are planning on going into the industry that uses a pre-installed operating system, Windows is the way to go!

Audio Players

Softwares that plays and organizes music or other audio

Rhythmbox

Rhythmbox is an audio player for Linux. Most commonly used with the GNOME interface. Rhythmbox features include;

  • Music Playback
  • Gapless Playback
  • Music Import
  • Audio CD Burning
  • Album Cover Display
  • Song Lyric Display
  • Web Remote Control
  • And More!

Windows Media Player

Windows Media Playeris an audio and video player for Windows. Auto-installed with most windows OS. WMP features include;

  • Playback, audio, videos, etc.
  • Visualizations
  • Format Support
  • Burning and Ripping
  • Shell Integration
  • Online Features
  • Media Streaming
  • And More!

Word Processing

For those who write and edit, word processing is a must.

LibreOffice

LibreOffice is the open soruce type of word processor. This open source software is most commonly used on Linux OS, but can be used on MacOS, and Windows.

Mircosoft Word

Microsoft Word, or more commonly known as just 'Word' is the microsoft version of a word processor. Microsoft Word comes bundled with the rest of the Mircosoft Office suite.

Internet Browser

These days, what's the use of a computer without the Internet?

Firefox

Firefox is the open source web browser. Being open source, the many Linux users prefer to use this brower if they don't want a text browser.

Microsoft Edge

Microsoft Edge is the latest and greatest version of Micosoft Internet Explorer. Edge has been updated and formatted for Windows 10. Edge is a default web brower for any Windows 10 machine.

W3M

W3M is another open source web browser. However, it is a text browser, so it works in basic HTML to load the page, instead of working with images and fancy formatting.

Google Chrome

Google Chrome is Google's version of a web browers. Chrome is compatable will most all devices and free to install.

Antivirus

Antivirus software is a good way to proctect your machine from attacks.

Linux Antivirus, what a joke

If properly configured, a Linux machine does not require Antivirus software, if it is for personal use and not server use. Linux software has few viruses that exist to penetrate the machine, most are aimed toward servers. Properly configuring your Linux PC will ensure that almost no virus or malware exists if you know what you are doing.

Windows Defender

Windows Defender is the free pre-installed Windows' version of antivirus software. Most times with antivirus, you get what you paid for, but with Defender it's free. While many people speculate the usefulness of this software, it's full capability has yet to be easily found.
